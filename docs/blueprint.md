# **App Name**: StreamVerse

## Core Features:

- Live Stream Playback: Play M3U8 (HLS) live streams and MP4 videos using an HTML5 video player.
- Channel Categories: Organize channels into categories like News, Sports, Movies, etc., with filtering and sorting options.
- Search Functionality: Enable users to search for channels by name, category, or keywords.
- Video Player Controls: Implement volume control, mute/unmute, full-screen mode, and picture-in-picture mode.
- Adaptive Bitrate Streaming: Dynamically adjust video quality based on the user's internet speed to ensure smooth playback.
- Gesture-Based Channel Switching: Use swipe gestures to quickly switch between channels.
- Channel Recommendation Tool: AI-powered recommendations of live TV streams and on-demand video content to users. The AI LLM uses a tool that assesses the user's viewing history to decide what to include in the recommendations.

## Style Guidelines:

- Primary color: Deep Indigo (#6660B2) for a modern and premium feel. 
- Background color: Very light gray (#F4F4F8), providing a clean and accessible base.
- Accent color: Violet (#B260AF), providing contrast to highlight the primary elements.
- Body font: 'PT Sans', a humanist sans-serif for general UI and readability.
- Headline font: 'Playfair', a modern sans-serif, for headers and titles.
- Use sleek, minimalist icons for channel categories and player controls.
- Employ a card-based layout for channels with thumbnails, logos, and live indicators.
- Implement subtle micro-animations for transitions and interactions to enhance user experience.
